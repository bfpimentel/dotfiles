# Pi agent configuration

When modifying my pi configuration, extensions, skills, prompts, themes, keybindings, or other pi agent files, use the directory from `PI_CODING_AGENT_DIR`.

On this machine, `PI_CODING_AGENT_DIR` is: `$HOME/.config/pi/agent`

Do not assume the default `~/.pi/agent` path unless `PI_CODING_AGENT_DIR` is unset.
